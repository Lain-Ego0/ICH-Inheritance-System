# ICH-Inheritance-System

## 非遗文化传承管理系统（SpringBoot+Vue）

### 项目简介

本项目是一套基于**SpringBoot（后端）+ Vue（前端）**的前后端分离式非遗文化传承管理系统，致力于通过数字化手段实现非物质文化遗产的保护、传承、推广与产业化发展。系统深度整合
**WebSocket即时通讯**、**协同过滤智能推荐算法**、**支付宝沙盒支付**及**社交分享链接**
等特色功能，为非遗传承人、爱好者、学习者构建一个集资源管理、互动交流、文化消费于一体的生态平台。

### 核心功能

1. **数字化非遗资源管理**
    - 构建非遗项目、传承人、技艺流程、文化作品的全维度数据库，支持图文、音视频、3D模型等多形态资源的上传与管理。
2. **WebSocket即时通讯**
    - 实现传承人之间、传承人和学习者之间的**实时消息沟通**，支持一对一聊天、群组讨论，助力非遗技艺的即时交流与经验分享。
3. **协同过滤智能推荐**
    - 基于用户行为与兴趣标签，通过**协同过滤算法**实现非遗项目、课程、周边产品的**个性化推荐**，提升用户文化探索的精准性与趣味性。
4. **支付宝沙盒支付**
    - 集成**支付宝沙盒支付**环境，支持非遗课程报名、文创产品购买等场景的在线支付，为非遗产业化提供技术支撑（适配开发与测试阶段的支付功能验证）。
5. **社交化分享传播**
    - 生成非遗项目、文化活动的**可分享链接**，支持一键分享至社交平台，助力非遗文化的裂变式传播，扩大受众覆盖面。
6. **多角色精细化管理**
    - 涵盖管理员、传承人、普通用户等角色，实现权限分级、资源审核、活动管理等全流程运营功能。

### 技术栈

- **后端**：SpringBoot + MyBatis Plus + WebSocket（即时通讯） + 协同过滤算法模块 + 支付宝沙盒SDK
- **前端**：Vue + Element UI + WebSocket客户端 + 社交分享组件
- **数据库**：MySQL
- **其他**：JWT权限认证 + Redis缓存 + Docker容器化部署

### 项目价值

本系统通过技术创新为非遗传承注入新活力：WebSocket打破时空限制，让技艺交流更高效；协同过滤让文化推荐更“懂”用户，降低非遗认知门槛；支付功能推动非遗从“保护”到“产业发展”的闭环；分享机制则让文化传播更具社交属性。最终实现“传承有工具、传播有效率、产业有支撑”的非遗生态建设目标。
